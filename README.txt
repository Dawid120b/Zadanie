Celem zadania jest napisanie prostej aplikacji korzystającej z OMDb API - filmowej otwartej bazy danych.
W trakcie wykonywania zadania skorzystaj z widoków załączonych w projekcie. Przygotowane są dwa projekty:

1. Wersja filmowa
2. Wersja serialowa

Aplikacja powinna sama dobierać layout ze względu na wyszukany kontent. W wyszukiwaniu zignoruj gry komputerowe.
Preferowany framework to AngularJS w wersji 1.5, jako preprocesora CSS uzyj SASS lub LESS.
Aplikacja powinna byc napisana z wykorzystaniem "graceful degradation" oraz RWD. Nie powinno być problemow z jej wyświetleniem na najnowszych przeglądarkach.

Nie ma projektu prezentacji wynikow wyszukiwania - ten widok chcielibyśmy abyś stworzyl sam. Moze to być wariacja na temat SELECT'a, a moze całkiem osobny widok - to nalezy do Ciebie.
Podobnie zresztą z Responsive Web Design - aplikacja powinna być dobrze prezentowana na urządzeniach mobilnych, jak i na desktopowych. Projektow mobilnych takze nie ma - liczymy na Ciebie.

Jezeli korzystasz z GIT'a, zaprezentuj nam aplikację właśnie poprzez repozytorium.

Trzymamy kciuki!
